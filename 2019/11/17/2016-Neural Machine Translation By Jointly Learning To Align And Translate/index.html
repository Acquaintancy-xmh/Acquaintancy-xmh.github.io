    • 2016-Neural Machine Translation By Jointly Learning To Align And Translate
    Created 2019-11-17| Updated 2019-11-17
    Post View:

    2019.8.25

    1. 最开始设计attention是为了解决encoder-decoder对长句子不友好的问题。

    2. 这里的attention（也就是alignment model 也就是标题的来源）是用decoder中hidden layer的state和encoder中hidden layer的state通过一个NN（也可以是别的方式）来获得权重的。得到权重后再和encoder中hidden layer相乘得到c。

      Transformer的Self-attention机制，直接把替代了这个模型的RNN部分。

    3. 很棒的一个想法是，attention的参数（alignment model的参数）可以和整个model一起train出来，也就是标题的“jointly”。

    4. 另一个改进是encoder中用了Bi-directional RNN，使得attention可以同时注意到前文和后文。

    Abstract

    1. The models proposed recently for neural machine translation often belong to a family of encoder–decoders and encode a source sentence into a ﬁxed-length vector from which a decoder generates a translation.
    2. In this paper, we conjecture that the use of a ﬁxed-length vector is a bottleneck in improving the performance of this basic encoder–decoder architecture, and propose to extend this by allowing a model to automatically (soft-)search for parts of a source sentence that are relevant to predicting a target word, without having to form these parts as a hard segment explicitly.
    1. 以前的encoder-decoder模型是把source sentence压缩成一个固定长度的vector，然后输入到decoder中产生新的序列。
    2. 这篇paper中，作者提出了这个固定长度的vector是限制model的瓶颈，所以提出了一个模型，这个模型能自动地找到source sentence中的和将要预测的word相关的词 – 即attention。
    3. paper中，attention这个词几乎没有出现，而是用alignment model来表示attention。

    Introduction

    1. An encoder neural network reads and encodes a source sentence into a fixed-length vector.

      A decoder then outputs a translation from the encoded vector.

      The whole encoder–decoder system, which consists of the encoder and the decoder for a language pair, is jointly trained to maximize the probability of a correct translation given a source sentence.

    2. ISSUE : the NN needs to be able to compress all the necessary information of a source sentence into a fixed-length vector . This may make it difﬁcult for the neural network to cope with long sentences, especially those that are longer than the sentences in the training corpus.

    3. TO ADRESS : Each time the proposed model generates a word in a translation ,it searches for a set of positions in a source sentence where the most relevant information is concentrated.

      The model then predicts a target word based on the context vectors associated with these source positions and all the previous generated target words.

    4. DISTINGUISHING FEATURE : It encodes the input sentence into a sequence of vectors and chooses a subset of these vectors adaptively while decoding the translation.

    1. 传统的encoder和decoder，是将encoder和decoder一起训练，为了最大化输出正确sentence的概率。
    2. 但是问题是encoder需要吧source sentence的所有的信息都压缩到vector中，所以当碰到长句子的时候会遇到问题。
    3. paper中的解决方法是输出每个词的时候，找到source sentence中和target word最有关的信息作为输入，然后加上上一个输出的词一起，得到这个词的输出。

    Background : Neural Machine Translation

    1. It reported that the neural machine translation based on RNNs with long short-term memory (LSTM) units achieves close to the state-of-the-art performance of the conventional phrase-based machine translation system on an English-to-French translation task.

    Learning to Align and Translate

    1. A novel architecture : consists of a bi-directional RNN as an encoder and a decoder that emulates searching through a source sentence during decoding a translation .

    创新点：1. 双向RNN 2. 引入attention机制（paper中叫Alignment model）

    Decoder

    1. As following :

    2. Note that unlike in traditional machine translation,the alignment is not considered to be a latent variable. Instead, the alignment model directly computes a soft alignment, which allows the gradient of the cost function to be backpropagated through. This gradient can be used to train the alignment model as well as the whole translation model jointly.

    3. ATTENTION : The probability αij , or its associated energy eij , reﬂects the importance of the annotation hj with respect to the previous hidden state si−1 in deciding the next state si and generating yi .

    1. 传统的encoder-decoder模型中，对齐（位置信息）是默认的，是隐藏的feature，但是在attention中，alignment信息是可以反向传播的，所以可以让alignment model和整个model一起train。
    2. attention的输出概率其实就是一个期望。

    ####Encoder :

    1. We would like the annotation of each word to summarize not only the preceding words, but also the following words. Hence, we propose to use a bidirectional RNN.
    1. encoder是用一个双向RNN，可以选用LSTM。

    Conclusion

    1. This has a major positive impact on the ability of the neural machine translation system to yield good results on longer sentences.
    2. One of challenges left for the future is to better handle unknown, or rare words. This will be required for the model to be more widely used and to match the performance of current state-of-the-art machine translation systems in all contexts.
    1. Attention机制能够很好的解决传统encoder-decoder模型中对长句子不友好的问题。
    2. 继续改进的方向是无法很好地处理未知词的问题。
